# Viacom

Viacom was an American mass media conglomerate that merged with CBS Corporation in December 2019 to form ViacomCBS (later rebranded as Paramount Global). The combined entity owned brands including CBS, MTV, Nickelodeon, Comedy Central, BET, Paramount Pictures, Showtime, and Paramount+. Paramount Global was subsequently acquired by Skydance Media and ceased to exist as an independent public company in August 2025.

## Overview

Viacom did not offer a public external developer API program. The company maintained open source software through its GitHub organization (ViacomInc), including tools for Roku channel development, tvOS UI components, Swift utilities, broadcast calendar functions, and JavaScript data processing utilities.
